# 🎬 Cloud-Based Video Streaming Platform

## 🚀 Project Overview

A scalable video streaming platform built using AWS cloud services. This application allows users to upload, store, and stream videos efficiently using a serverless architecture.

The system eliminates the need for traditional servers and ensures high availability, scalability, and cost optimization.

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** AWS Lambda (Serverless)
* **Storage:** Amazon S3
* **API Management:** Amazon API Gateway
* **Security:** IAM Roles & Policies
* **Version Control:** Git & GitHub

---

## 🔥 Key Features

* 📤 Upload videos to AWS S3
* 🎥 Stream videos directly from cloud storage
* ⚡ Serverless backend using AWS Lambda
* 🔐 Secure access using IAM policies
* 🌐 REST API integration using API Gateway

---

## 🧩 System Architecture

User → React Frontend → API Gateway → AWS Lambda → S3 Bucket → Video Streaming

---

## 🧠 Architecture Explanation

1. User interacts with React frontend
2. API Gateway receives HTTP requests
3. Lambda function processes requests
4. Videos are stored/retrieved from S3
5. S3 serves video content for streaming
